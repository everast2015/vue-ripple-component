# 下载的地址
```
https://github.com/yjn2015/vue-ripple-component.git
```

## 安装相应的依赖
```
npm install
```

### 跑起来
```
npm run serve
```

### 生产环境使用
```
npm run build
```

### 校验
```
npm run lint
```
### 安装的插件 --save
> vue2-animate 是做动画的一个插件

1. npm install element-ui --save
2. npm install vue2-animate --save
